# 🏠 인테리어 멀티 에이전트 시스템

Google Agent Development Kit(ADK)를 사용해서 만든 **인테리어 업무를 자동화해주는 AI 로봇** 시스템입니다!

## 📁 프로젝트 파일 구조

```
interior_multi_agent/                   # 📂 메인 프로젝트 폴더
├── 📂 interior_agents/                 # AI 에이전트들이 모여있는 폴더
│   ├── 📄 __init__.py                  # Python 패키지 초기화 파일
│   ├── 🤖 agent.py                     # 모든 AI 에이전트들의 핵심 코드
│   ├── 🔑 .env                         # API 키와 비밀 설정들
│   └── 📝 config_template.txt          # 설정 파일 만들기 도움말
├── 📂 .venv/                           # 가상환경 폴더 (자동 생성됨)
├── 📄 requirements.txt                 # 필요한 프로그램 목록
└── 📖 README.md                        # 사용법 설명서 (이 파일!)
```

## 🤖 각 파일이 하는 일

### 📄 `__init__.py` 
```python
from . import agent
```
- **역할**: Python이 이 폴더를 하나의 패키지로 인식하게 해주는 파일
- **쉽게 말하면**: "이 폴더 안에 중요한 코드들이 있어!" 라고 알려주는 표지판

### 🤖 `agent.py` (12KB - 핵심 파일!)
이 파일 안에는 **4명의 전문가 AI 로봇**들이 들어있습니다:

#### 🏢 1. 현장주소 관리 로봇
```python
# 이런 일들을 담당해요:
def register_site()      # 새 공사현장 정보 저장하기
def get_site_info()      # 현장 정보 찾아보기  
def list_all_sites()     # 모든 현장 목록 보여주기
```
- **하는 일**: 공사 현장의 주소, 크기 등 기본 정보를 기록하고 관리
- **예시**: "강남구 아파트 30평, 주소는 xxx" 이런 정보들을 저장

#### 💰 2. 원가계산 로봇
```python  
# 이런 일들을 담당해요:
def calculate_material_cost()  # 자재(재료) 비용 계산하기
def estimate_labor_cost()      # 일하는 사람 인건비 계산하기
def get_total_cost()          # 전체 비용 합계 내기
```
- **하는 일**: 페인트, 타일 등 재료비와 일하는 사람 비용을 자동으로 계산
- **예시**: "타일 30㎡ × 30,000원 = 900,000원" 이런 계산들

#### 💳 3. 결제관리 로봇
```python
# 이런 일들을 담당해요:
def create_payment_invoice()   # 청구서(계산서) 만들기
def get_payment_status()       # 돈 받았는지 확인하기
def update_payment_status()    # 결제 상태 바꾸기
def list_all_invoices()        # 모든 청구서 목록 보기
```
- **하는 일**: 고객한테 보낼 계산서를 만들고, 돈을 받았는지 관리
- **예시**: "GN001 현장 총 300만원, 아직 미결제" 이런 상태 관리

#### 📊 4. 프로젝트 현황관리 로봇
```python
# 이런 일들을 담당해요:
def get_project_summary()      # 전체 사업 현황 요약하기
```
- **하는 일**: 진행중인 모든 프로젝트의 통계와 수익 현황을 한눈에 보여줌
- **예시**: "총 5개 현장, 수익 1500만원, 미수금 300만원" 같은 요약

### 🔑 `.env` 파일
```bash
GOOGLE_GENAI_USE_VERTEXAI=FALSE
GOOGLE_API_KEY=여기에_실제_키_입력
```
- **역할**: Google AI를 사용하기 위한 비밀번호(API 키) 저장
- **중요**: 이 파일은 **절대 남에게 보여주면 안됨** (비밀번호니까!)

### 📝 `config_template.txt`
- **역할**: `.env` 파일을 올바르게 만들 수 있도록 도와주는 견본
- **사용법**: 이 파일을 복사해서 `.env`로 이름 바꾸고 API 키만 입력하면 됨

### 📄 `requirements.txt`
```
google-adk
python-dotenv
```
- **역할**: 이 프로그램이 작동하는데 필요한 다른 프로그램들의 목록
- **쉽게 말하면**: "이 재료들이 있어야 요리(프로그램)를 만들 수 있어요" 하는 장보기 목록

## 📋 설치하고 실행하는 방법

### 🔧 1단계: 컴퓨터 환경 준비하기
```bash
# 1. 프로그램들끼리 충돌 안나게 격리된 공간(가상환경) 만들기
python -m venv .venv

# 2. 가상환경 켜기
# Windows 사용자:
.venv\Scripts\activate
# Mac/Linux 사용자:
source .venv/bin/activate

# 3. 필요한 프로그램들 설치하기
pip install -r requirements.txt
```

### 🗝️ 2단계: Google AI 사용 권한 받기

#### Google AI Studio에서 무료 API 키 받기:
1. 🌐 https://aistudio.google.com/ 접속
2. 🔑 **"Get API Key"** 버튼 클릭
3. 📋 생성된 키 복사하기

#### 설정 파일에 키 입력하기:
1. `interior_agents/config_template.txt` 파일을 복사
2. 이름을 `interior_agents/.env`로 바꾸기
3. 파일 열어서 API 키 입력:
```bash
GOOGLE_GENAI_USE_VERTEXAI=FALSE
GOOGLE_API_KEY=여기에_복사한_실제_API_키_붙여넣기
```

### 🚀 3단계: 프로그램 실행하기

#### 방법 1: 웹 브라우저에서 사용 (추천!)
```bash
adk web
```
- 브라우저에서 `http://localhost:8000` 접속
- 예쁜 화면에서 AI와 대화하면서 사용 가능

#### 방법 2: 터미널에서 바로 사용
```bash
adk run interior_agents
```
- 검은 화면에서 글자로만 AI와 대화

## ⚠️ 문제 해결 (뭔가 안될 때)

### 🚫 `adk: command not found` 에러가 날 때
- **원인**: 가상환경이 꺼져있거나 패키지 설치가 안됨
- **해결방법**: 
  ```bash
  # 1. 가상환경 다시 켜기
  .venv\Scripts\activate  # Windows
  source .venv/bin/activate  # Mac/Linux
  
  # 2. 패키지 다시 설치
  pip install -r requirements.txt
  ```

### 🔑 API 키 관련 에러가 날 때
- **원인**: `.env` 파일에 올바른 API 키가 없음
- **해결방법**: 
  1. Google AI Studio에서 새 API 키 발급
  2. `.env` 파일 다시 확인하고 올바른 키 입력

### 🌐 웹 UI가 안 열릴 때
- **해결방법**: `adk web --no-reload` 명령어 사용
